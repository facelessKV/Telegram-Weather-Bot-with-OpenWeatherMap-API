=== КАК ЗАПУСТИТЬ ТЕЛЕГРАМ БОТА ПОГОДЫ ===

Это пошаговая инструкция для запуска Телеграм-бота погоды на Windows и Linux.
Следуйте инструкциям для вашей операционной системы.

=====================================================================
                         WINDOWS
=====================================================================

1. УСТАНОВКА PYTHON (ВЕРСИЯ 3.10)
--------------------------------
- Перейдите на сайт: https://www.python.org/downloads/release/python-31011/
- Прокрутите страницу вниз и скачайте файл "Windows installer (64-bit)"
- Запустите скачанный файл
- ВАЖНО: Поставьте галочку "Add Python 3.10 to PATH" при установке!
- Нажмите "Install Now" и дождитесь окончания установки

2. ПРОВЕРКА УСТАНОВКИ PYTHON
---------------------------
- Нажмите на клавиатуре Win + R
- Напишите "cmd" и нажмите Enter
- В открывшейся командной строке введите:
  python --version
- Вы должны увидеть что-то вроде "Python 3.10.11"

3. ПОЛУЧЕНИЕ API КЛЮЧЕЙ
----------------------
a) Получение Telegram Bot Token:
   - Откройте Telegram
   - Найдите @BotFather и напишите ему
   - Отправьте команду /newbot
   - Следуйте инструкциям, чтобы дать имя вашему боту
   - Сохраните полученный токен (выглядит как 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11)

б) Получение OpenWeatherMap API Key:
   - Перейдите на сайт https://openweathermap.org/
   - Нажмите "Sign Up" и создайте аккаунт
   - После регистрации перейдите в "My API keys"
   - Сохраните API ключ (или создайте новый)

4. СКАЧИВАНИЕ ФАЙЛОВ БОТА
------------------------
- Создайте новую папку на вашем компьютере, например, "weather-bot"
- Сохраните файлы main.py, config.py и README.md в эту папку

5. НАСТРОЙКА БОТА
---------------
- Откройте файл config.py с помощью блокнота (кликните правой кнопкой мыши по файлу -> Открыть с помощью -> Блокнот)
- Замените "YOUR_TELEGRAM_BOT_TOKEN" на токен, полученный от @BotFather
- Замените "YOUR_OPENWEATHERMAP_API_KEY" на ключ от OpenWeatherMap
- Сохраните файл (Ctrl+S)

6. УСТАНОВКА ЗАВИСИМОСТЕЙ
-----------------------
- Вернитесь в командную строку (или откройте новую: Win + R -> cmd -> Enter)
- Введите команду для перехода в папку с файлами бота:
  cd путь_к_вашей_папке
  
  Например:
  cd C:\Users\ИмяПользователя\Documents\weather-bot
  
- Установите необходимые библиотеки командой:
  pip install aiogram==3.0.0 aiohttp

7. ЗАПУСК БОТА
------------
- В командной строке введите:
  python main.py
- Если всё сделано правильно, вы увидите сообщения о запуске бота
- Бот будет работать, пока командная строка открыта

8. ИСПОЛЬЗОВАНИЕ БОТА
------------------
- Откройте Telegram
- Найдите своего бота по имени, которое вы дали при создании
- Отправьте команду /start
- Чтобы узнать погоду, напишите "погода" и название города (например, "погода Москва")

=====================================================================
                            LINUX
=====================================================================

1. УСТАНОВКА PYTHON (ВЕРСИЯ 3.10)
--------------------------------
- Откройте терминал (Ctrl+Alt+T в большинстве дистрибутивов)
- Введите следующие команды:

  Для Ubuntu/Debian:
  sudo apt update
  sudo apt install software-properties-common
  sudo add-apt-repository ppa:deadsnakes/ppa
  sudo apt update
  sudo apt install python3.10 python3.10-venv python3-pip

  Для Fedora:
  sudo dnf install python3.10

  Для CentOS/RHEL:
  sudo yum install python3.10

2. ПРОВЕРКА УСТАНОВКИ PYTHON
---------------------------
- В терминале введите:
  python3.10 --version
- Вы должны увидеть "Python 3.10.x"

3. ПОЛУЧЕНИЕ API КЛЮЧЕЙ
----------------------
a) Получение Telegram Bot Token:
   - Откройте Telegram
   - Найдите @BotFather и напишите ему
   - Отправьте команду /newbot
   - Следуйте инструкциям, чтобы дать имя вашему боту
   - Сохраните полученный токен

б) Получение OpenWeatherMap API Key:
   - Перейдите на сайт https://openweathermap.org/
   - Нажмите "Sign Up" и создайте аккаунт
   - После регистрации перейдите в "My API keys"
   - Сохраните API ключ (или создайте новый)

4. СКАЧИВАНИЕ ФАЙЛОВ БОТА
------------------------
- Создайте новую папку командой:
  mkdir ~/weather-bot
- Перейдите в папку:
  cd ~/weather-bot
- Создайте файлы командами:
  nano main.py
  (вставьте код из main.py, затем Ctrl+O для сохранения, Enter, и Ctrl+X для выхода)
  
  nano config.py
  (вставьте код из config.py, затем Ctrl+O для сохранения, Enter, и Ctrl+X для выхода)

5. НАСТРОЙКА БОТА
---------------
- Откройте config.py:
  nano config.py
- Замените "YOUR_TELEGRAM_BOT_TOKEN" на токен, полученный от @BotFather
- Замените "YOUR_OPENWEATHERMAP_API_KEY" на ключ от OpenWeatherMap
- Сохраните файл: Ctrl+O, затем Enter, затем Ctrl+X

6. СОЗДАНИЕ ВИРТУАЛЬНОГО ОКРУЖЕНИЯ И УСТАНОВКА ЗАВИСИМОСТЕЙ
--------------------------------------------------------
- В терминале выполните команды:
  python3.10 -m venv venv
  source venv/bin/activate
  pip install aiogram==3.0.0 aiohttp

7. ЗАПУСК БОТА
------------
- В терминале, убедитесь что вы активировали виртуальное окружение (должно быть (venv) в начале строки)
- Если нет, активируйте его:
  source venv/bin/activate
- Запустите бота:
  python main.py
- Бот будет работать, пока терминал открыт

8. ЗАПУСК БОТА В ФОНОВОМ РЕЖИМЕ (ОПЦИОНАЛЬНО)
-------------------------------------------
- Чтобы бот не останавливался при закрытии терминала, выполните:
  nohup python main.py > bot_log.txt 2>&1 &
- Чтобы проверить, запущен ли бот:
  ps aux | grep python

9. ИСПОЛЬЗОВАНИЕ БОТА
------------------
- Откройте Telegram
- Найдите своего бота по имени, которое вы дали при создании
- Отправьте команду /start
- Чтобы узнать погоду, напишите "погода" и название города (например, "погода Москва")

=====================================================================
          ВОЗМОЖНЫЕ ОШИБКИ И ИХ РЕШЕНИЯ
=====================================================================

1. "python не является внутренней или внешней командой..." (Windows)
   Решение: Убедитесь, что вы поставили галочку "Add Python to PATH" при установке.
   Альтернативно, используйте полный путь к Python или переустановите Python.

2. "ModuleNotFoundError: No module named 'aiogram'"
   Решение: Убедитесь, что вы правильно установили зависимости командой pip install.

3. "Не удалось подключиться к боту"
   Решение: Проверьте интернет-соединение и правильность Telegram Bot Token.

4. "Ошибка при получении данных о погоде"
   Решение: Проверьте правильность OpenWeatherMap API Key и его активацию.

5. Для Linux: "Command 'python' not found"
   Решение: Используйте команду python3 или python3.10 вместо python.
